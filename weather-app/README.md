# Weather App

This is a minimal weather app that displays live weather data for a city using the OpenWeatherMap API. 

## Features
- Shows only the current weather and temperature
- Uses live data (not random)
- Simple, clean UI

## Usage
1. Replace `YOUR_API_KEY` in `index.html` with your OpenWeatherMap API key.
2. Open the page in your browser or host it on GitHub Pages.
